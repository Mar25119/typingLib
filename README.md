# typingLib

# Проект: Фильтрация пользователей с аннотациями типов

Этот небольшой учебный проект демонстрирует применение модуля `typing` из стандартной библиотеки Python для добавления аннотаций типов к функциям и данным.  
Основная цель — показать, как явное указание типов улучшает читаемость, поддерживаемость и надёжность кода.

## Задача

Реализовать функцию `filter_adults`, которая принимает список пользователей (каждый — словарь с полями `name` и `age`) и возвращает только тех, кому 18 лет и старше.  
Весь код должен быть полностью аннотирован с использованием `typing`.

## Требования

- Python 3.9 или новее  
- (Опционально) `mypy` для статической проверки типов:
  ```bash
  pip install mypy
  ```

## Запуск

1. Склонируйте репозиторий или скопируйте файл `users_filter.py`.
2. Запустите программу:
   ```bash
   python users_filter.py
   ```
3. Вы увидите список совершеннолетних пользователей.

Пример вывода:
```
Совершеннолетние пользователи:
- Алиса, 25 лет
- Вася, 30 лет
```

## Проверка типов (опционально)

Для проверки корректности аннотаций выполните:
```bash
mypy users_filter.py
```
Если всё сделано правильно, mypy не выдаст ошибок.


## Используемые возможности `typing`

- `List[...]` — типизированный список  
- `Dict[...]` — типизированный словарь  
- `|` — объединение типов  
- Человекочитаемые сигнатуры функций

## Почему это важно?

Аннотации типов:
- Помогают IDE предлагать точные подсказки
- Упрощают чтение кода коллегами
- Позволяют находить ошибки до запуска программы
- Являются частью современных стандартов Python-разработки (PEP 484)
