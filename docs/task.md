# Техническое задание

## 1. Описание задачи
Разработка небольшой утилиты для обработки списка пользователей, где каждый пользователь представлен словарём с полями `name` (строка) и `age` (целое число). Необходимо добавить аннотации типов ко всем функциям и переменным.

## 2. Цель работы
Продемонстрировать понимание базового использования модуля `typing` для повышения читаемости и надёжности кода.

## 3. Функциональные требования
- Программа должна содержать функцию `filter_adults(users)`, возвращающую только пользователей старше 18 лет.
- Все параметры и возвращаемые значения должны быть аннотированы с использованием `typing` (например, `List[Dict[str, int]]`).
- Должен быть пример вызова функции с тестовыми данными.

## 4. Нефункциональные требования
- Код должен быть совместим с Python 3.9+.
- Аннотации должны быть корректными и соответствовать фактическому поведению функций.
- Код должен проходить проверку через `mypy` без ошибок.

## 5. Ограничения и допущения
- Нельзя использовать сторонние библиотеки (только стандартная библиотека, включая `typing`).
- Предполагается, что все входные данные корректны (валидация не требуется).
- Использование `from __future__ import annotations` не требуется.

## 6. Источник ТЗ
Самостоятельная постановка.
